CONNECTLY

Connectly is a digital relationship memory system designed to help professionals remember and manage the people they meet.

🚀 Features
- Private dashboard for storing contacts
- Profile sharing system
- Fast and responsive UI
- Secure authentication
- Supabase backend integration

🛠 Tech Stack
- React
- TypeScript
- Tailwind CSS
- Supabase
- Vite

📦 Installation

git clone https://github.com/Jaswanth01234/connectly
cd connectly
npm install
npm run dev

🎯 Use Case

Professionals meet many people every day. Connectly helps store and remember those connections in a structured dashboard.
